# 文化祭 Webサイト

文化祭向け Webサイトの学習・開発用土台です。  
デザインや機能はこれから少しずつ追加していく想定で、現時点では最小限の構成のみ用意しています。

## 使用技術

- HTML
- CSS
- JavaScript（必要最低限）
- ライブラリ・フレームワークは使用しません

## ディレクトリ構成

```
/
├── index.html              # トップページ
├── css/
│   └── style.css           # 共通スタイル
├── js/
│   └── script.js           # 共通スクリプト（現状は空）
├── images/                 # 画像置き場
├── pages/
│   ├── events.html         # 企画一覧ページ
│   └── timetable.html      # タイムスケジュールページ
└── README.md
```

## ページの開き方（URL）

| ページ | ファイル | 開き方の例 |
| ------ | -------- | ---------- |
| トップ | `index.html` | フォルダの `index.html` を開く |
| 企画一覧 | `pages/events.html` | ナビの「企画一覧」をクリック |
| タイムスケジュール | `pages/timetable.html` | ナビの「タイムスケジュール」をクリック |

同じページ内のジャンプ（`#events` など）ではなく、**別ページのURLへそのまま移動**する構成です。

## トップページの構成（index.html）

| セクション | ID        | 用途（想定）       |
| ---------- | --------- | ------------------ |
| Header     | `#header` | サイト名・ナビ     |
| Hero       | `#hero`   | メインビジュアル   |
| News       | `#news`   | お知らせ           |
| About      | `#about`  | 文化祭の概要       |
| Access     | `#access` | アクセス情報       |
| FAQ        | `#faq`    | よくある質問       |
| Footer     | `#footer` | フッター           |

## 内容の書き方（初心者向け）

JavaScript は不要です。該当の HTML を開いて、コメントどおりにコピー＆書き換えでOKです。

### 企画一覧を増やす

1. `pages/events.html` を開く
2. 「企画カード」と書いてある `<li>...</li>` の塊をコピー
3. 企画名・カテゴリ・場所・紹介文を書き換える

### タイムスケジュールを増やす

1. `pages/timetable.html` を開く
2. 表の1行分（`<tr>...</tr>`）をコピー
3. 時間・内容・場所を書き換える
4. 2日開催なら、コメント内の「2日目」の例を参考に表を追加

## ローカルでの確認方法

1. このフォルダを開く
2. `index.html` をブラウザで開く
3. 上部メニューから各ページへ移動できることを確認する

（任意）Live Server などの拡張機能を使うと、保存のたびに自動で再読み込みできます。

## 今後の開発メモ

- `pages/` にサブページを追加するときは、ナビのリンクも合わせて増やす
- `images/` に写真・ロゴを配置する
- `css/style.css` の `:root` で色や余白をまとめて調整する
- `js/script.js` にメニュー・FAQ などの挙動を追加する

## 方針

- semantic HTML（`header` / `main` / `section` / `footer`）を使う
- コメントを多めに残し、後から読みやすい状態を保つ
- 見た目より、保守性・拡張性を優先する
