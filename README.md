# 文化祭 Webサイト

文化祭向け Webサイトの学習・開発用土台です。  
デザインや機能はこれから少しずつ追加していく想定で、現時点では最小限の構成のみ用意しています。

## 使用技術

- HTML
- CSS
- JavaScript（必要最低限）
- ライブラリ・フレームワークは使用しません

## ディレクトリ構成

```
/
├── index.html          # トップページ
├── css/
│   └── style.css       # 共通スタイル
├── js/
│   └── script.js       # 共通スクリプト（現状は空）
├── images/             # 画像置き場
├── pages/              # サブページ置き場（今後追加）
└── README.md
```

## ページ構成（index.html）

| セクション | ID           | 用途（想定）               |
| ---------- | ------------ | -------------------------- |
| Header     | `#header`    | サイト名・ナビ（タブ）     |
| Hero       | `#hero`      | メインビジュアル           |
| News       | `#news`      | お知らせ                   |
| About      | `#about`     | 文化祭の概要               |
| 企画一覧   | `#events`    | 企画・ブース一覧           |
| タイムスケジュール | `#timetable` | 当日の時間表     |
| Access     | `#access`    | アクセス情報               |
| FAQ        | `#faq`       | よくある質問               |
| Footer     | `#footer`    | フッター                   |

ヘッダーのタブからは **企画一覧** と **タイムスケジュール** に移動できます。

## 内容の書き方（初心者向け）

どちらも `index.html` を開いて、コメントの指示どおりにコピー＆書き換えでOKです。JavaScript は不要です。

### 企画一覧を増やす

1. `id="events"` のセクションを探す
2. 「企画カード」と書いてある `<article>...</article>` の塊をコピー
3. 企画名・カテゴリ・場所・紹介文を書き換える

### タイムスケジュールを増やす

1. `id="timetable"` のセクションを探す
2. 表の1行分（`<tr>...</tr>`）をコピー
3. 時間・内容・場所を書き換える
4. 2日開催なら、コメント内の「2日目」の例を参考に表を追加

## ローカルでの確認方法

1. このフォルダを開く
2. `index.html` をブラウザで開く

（任意）Live Server などの拡張機能を使うと、保存のたびに自動で再読み込みできます。

## 今後の開発メモ

- `pages/` にサブページ（例: ブース詳細）を追加する
- `images/` に写真・ロゴを配置する
- `css/style.css` の `:root` で色や余白をまとめて調整する
- `js/script.js` にメニュー・FAQ などの挙動を追加する

## 方針

- semantic HTML（`header` / `main` / `section` / `footer`）を使う
- コメントを多めに残し、後から読みやすい状態を保つ
- 見た目より、保守性・拡張性を優先する
